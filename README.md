# HJS-SAX2
> NodeJS Sax2 driver for XML pull parser implementation (hjs-xmlpull and hjs-kxml2).

SAX2 classes: InputSource, DefaultHandler, SaxDriver.

## Installation

Node:

```sh
npm install hjs-sax2 --save
```
